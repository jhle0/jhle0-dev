# LLM 평가 루틴 메모

모델 결과를 비교할 때는 느낌보다 기준이 필요하다.

## 어떤 항목을 볼 것인가

정확성, 일관성, 형식 준수, 재현 가능성을 기준으로 본다.

## 프롬프트 실험 기록 방식

작은 변화라도 입력과 결과를 함께 남겨야 나중에 비교가 가능하다.
